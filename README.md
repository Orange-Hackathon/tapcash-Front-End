## ğŸ Getting started

1. **_Clone the repository_**

```sh
$ git clone https://github.com/Orange-Hackathon/Front-End.git
```

2. **_Navigate to repository directory_**

```sh
$ cd Front-End
```

3. **_Install dependencies_**

```sh
$ npm install
```

### Running

**_Running on development mode_**

```sh
$ npm start
```
